About Passenger Screening

Passenger Screening is An Machine Learning Algorithm to detect threats from passengers created by EC601 Team10. It is a Kaggle Project from U.S. Transportation Security Administration (TSA).

This Project maybe use Tensorflow and Amazon EC2 with Flaskas the platform.

Passenger Screening includes machine learning part and web application part. You can check both of them on our github
